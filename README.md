# 🌗 Dual World — Two Worlds. One Click.

An immersive and interactive web experience that lets users switch between two completely different UI realities — a clean, bright **Normal World** and a dark, glitchy **Upside Down** dimension.

---

## 🌐 Live Demo

👉 https://Akshat-gupta-01.github.io/Dual-World/

---

## ✨ Features

* 🌗 **Dual Mode Toggle**

  * Seamlessly switch between Normal and Upside Down worlds
* 🎨 **Dynamic UI Transformation**

  * Colors, layout, and atmosphere change instantly
* ⚡ **Smooth Animations**

  * CSS-based transitions and flipping effects
* 🎇 **Particle System**

  * Interactive animated particles with mode-based behavior
* 👾 **Glitch Effects**

  * Real-time glitch overlays and distorted UI in dark mode
* 🔊 **Immersive Audio**

  * Ambient soundscapes for each world using Web Audio API

---

## 🧠 Concept

Dual World is inspired by the idea of parallel dimensions — where a single interface can transform into a completely different emotional experience with just one click.

* ☀️ **Normal World**

  * Clean, minimal, and user-friendly design
* 🌑 **Upside Down**

  * Dark, eerie, glitch-heavy immersive UI

---

## 🛠️ Tech Stack

* **HTML5** – Structure 
* **CSS3** – Styling & animations 
* **JavaScript** – Logic & interactivity 
* **Web Audio API** – Sound effects
* **Canvas API** – Particle system
